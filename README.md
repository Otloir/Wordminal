## Installation

### clone the repo

git clone https://github.com/Otloir/Lingon.git

### install dependencies

#### Installing NPM:

`
npm init -i
`

#### Packages:

Chalk:

`
npm install chalk
`


Inquirer

`
npm install @inquirer/prompts
`


figlet

`
npm install figlet
`


MongoDB
